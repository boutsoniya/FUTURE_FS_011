## 🔍 About The Project

This is my professional portfolio website, designed and developed as a part of the **Future Interns - Full Stack Web Development Task 1**.

It serves as my digital resume and a centralized hub to showcase my skills, projects, and professional background. Rather than just fulfilling an internship task, this was built entirely from scratch with a focus on modern web design, high performance, and elegant user experience—serving as the first impression for recruiters and freelance clients.

### ✨ Key Features
- **Modern Aesthetics**: Sleek dark mode with glassmorphism effects, dynamic gradients, and smooth hover states.
- **Fully Responsive**: Built with modern CSS Grid and Flexbox to ensure it looks phenomenal on desktops, tablets, and smartphones.
- **Interactive Elements**: Features scroll-reveal animations, dynamic sticky navigation, and a simulated contact form using JavaScript.
- **SEO Optimized**: Includes proper semantic HTML structure and meta tags for improved search engine visibility.

## 🛠️ Tech Stack

This project was built entirely using Vanilla web technologies to ensure maximum performance and minimal dependency bloat:
- **HTML5**: For semantic and accessible structure.
- **CSS3 (Vanilla)**: For advanced styling, variables, and responsive design (no external frameworks used).
- **JavaScript (ES6+)**: For DOM manipulation, intersection observers, and smooth scrolling logic.

## 🚀 Getting Started (Run Locally)

Since this project uses no build tools or package managers, running it locally is incredibly simple.

### Option 1: Live Server (Recommended)
1. Open this repository in **VS Code**.
2. Install the **Live Server** extension.
3. Right-click on `index.html` and select **"Open with Live Server"**.

### Option 2: Python HTTP Server
If you have Python installed, you can quickly spin up a local server:
```bash
python -m http.server 8080
```
Then visit `http://localhost:8080/` in your browser.

### Option 3: Direct File Open
Simply double-click the `index.html` file in your system's file explorer.
